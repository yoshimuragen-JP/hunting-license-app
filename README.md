## 📖 概要

このアプリは、狩猟免許試験（特に第一種銃猟免許）の合格を目指す方のための学習ツールです。

### 特徴

- **完全無料**: 全機能を無料で利用可能（広告なし、課金なし、登録不要）
- **網羅的**: 法令、猟具、鳥獣識別、保護管理、実技まで完全カバー
- **実践的**: 220問以上の厳選問題、3回分の模擬試験、詳細な解説付き
- **分析機能**: カテゴリ別正答率、弱点分析、合格予測
- **データ管理**: エクスポート/インポート機能でバックアップ・復元可能
- **PWA対応**: スマホにインストールしてアプリのように使える
- **オフライン対応**: ネット環境なしでも学習可能

### 誰のためのアプリか

- これから狩猟免許を取得したい方
- 独学で効率的に学習したい方
- 鳥獣識別に不安がある方
- 実技試験の対策をしたい方
- 模擬試験で実力を測りたい方

## 🚀 使い方

### インストール方法（PWA）

#### スマートフォン（iPhone/Android）

1. SafariまたはChromeでアプリを開く
2. **iPhone**: 共有ボタン → 「ホーム画面に追加」
3. **Android**: メニュー（︙） → 「ホーム画面に追加」
4. アイコンをタップして起動

#### パソコン（Chrome/Edge）

1. アドレスバーのインストールボタン（＋アイコン）をクリック
2. 「インストール」をクリック
3. デスクトップまたはスタートメニューから起動

### 学習の進め方

#### ステップ1: 基礎学習で全体像を把握

まずは5つのカテゴリ（法令、猟具、鳥獣識別、保護管理、実技）の基礎知識を学びましょう。

#### ステップ2: 問題演習で知識を定着

カテゴリ別またはランダムで問題を解き、理解を深めます。間違えた問題は復習機能で重点的に学習できます。

#### ステップ3: 鳥獣図鑑で識別力UP

30種以上の鳥獣の特徴、識別ポイントを学習。狩猟鳥獣と非狩猟鳥獣を確実に見分けられるようにします。

#### ステップ4: 実技ガイドで実践力養成

銃器やわなの取り扱い、安全点検の手順をマスター。実技試験で減点されないためのポイントを確認します。

#### ステップ5: 模擬試験で総仕上げ

本番と同じ形式・時間制限で実力を測定。70%以上を安定して取れるようになれば合格圏内です。

### 推奨学習フロー

```
基礎学習（3-5日） → 問題演習（7-10日） → 鳥獣図鑑（2-3日） → 実技ガイド（2-3日） → 模擬試験（3回）
                                ↓
                          弱点カテゴリを集中復習
                                ↓
                            本番試験へ
```

**推奨学習時間**: 合計20時間以上
**推奨問題数**: 全220問を最低2周（計440問以上）

## 📚 機能一覧

### 1. 問題演習

- **220問以上**の厳選された過去問・予想問題を収録
- カテゴリ別・ランダム出題
- 詳細な解説付き
- 間違えた問題の復習機能
- 正答率・弱点分析

### 2. 模擬試験

- 本番と同じ形式（30問、三肢択一）
- 制限時間90分
- 合格判定（70%ライン）
- 3回分の完全模擬試験
- カテゴリ別成績分析

### 3. 鳥獣図鑑

- **43種**の鳥獣を詳細解説
- 狩猟鳥獣と非狩猟鳥獣の識別ポイント
- 写真・イラスト・特徴の3点セット
- 外観比較クイズ

### 4. 実技ガイド

- 猟銃の取り扱い・点検手順
- わなの設置・取り扱い方法
- 安全管理チェックリスト
- よくある失敗例と対策
- 減点ポイントの解説

### 5. 進捗管理

- 総合正答率
- カテゴリ別正答率
- 累計学習時間
- 総解答数
- 合格可能性予測
- 弱点カテゴリの自動検出

### 6. データ管理機能

- **データエクスポート**: 学習データをJSON形式でバックアップ
- **データインポート**: バックアップからデータを復元
- **データリセット**: 学習データを初期化（二重確認あり）
- **トースト通知**: 操作完了時に視覚的なフィードバック
- **複数デバイス対応**: エクスポート/インポートでデータ移行可能

### 7. ゲームモード

- クイックマッチ（5問スピードクイズ）
- デイリーチャレンジ（毎日10問）
- 連続正解コンボシステム
- バッジ・実績獲得（14種類）
- ハイスコアランキング

### オフライン対応

- 鳥獣図鑑
- 実技ガイド
- 進捗確認

**注意**: 初回アクセス時はインターネット接続が必要です。

### SEO対応

Webアプリとして公開する際の検索エンジン最適化：

- **robots.txt**: クローラーの制御設定
- **sitemap.xml**: 全ページの構造マップ
- **メタタグ**: 各ページに適切なメタ情報を設定

### データ管理

学習データはブラウザの**LocalStorage**に保存されます。

#### 保存されるデータ

- 問題の解答履歴
- 正答率・学習時間
- カテゴリ別成績
- 間違えた問題のリスト

#### データの保持

- データはブラウザに保存されるため、**アカウント登録不要**
- ブラウザのデータを削除すると学習データも消えます
- 複数デバイス間でのデータ同期は現在非対応

#### データのエクスポート・インポート

**学習データをJSON形式でエクスポート・インポートできます。**

- **エクスポート**: 学習データを`huntingProgress_backup_YYYYMMDD.json`形式でダウンロード
- **インポート**: バックアップファイルを選択して学習データを復元
- **用途**:
  - デバイス間でのデータ移行（スマホ⇔PC）
  - 学習データのバックアップ
  - 端末変更時のデータ引き継ぎ

**注意**: インポートすると既存のデータは上書きされます。事前にエクスポートでバックアップを取ることをおすすめします。
