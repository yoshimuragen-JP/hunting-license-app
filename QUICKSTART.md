## 📱 ステップ1: アプリを開く（1分）


### スマホにインストール（おすすめ）

**iPhone**:
1. Safariで開く
2. 画面下の「共有」ボタン（□に↑）をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面のアイコンから起動

**Android**:
1. Chromeで開く
2. 右上のメニュー（︙）をタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面のアイコンから起動

**インストールのメリット**:
- アプリのように使える
- オフラインでも学習できる
- 通知機能（今後実装予定）

## 📚 ステップ3: 機能を使いこなす（2分）

### 主要な6つの機能

#### 1. 問題演習（最重要）

```
問題演習ボタン → カテゴリ選択 → 問題を解く → 解説を読む
```

**ポイント**:
- まずは「カテゴリ別」で各カテゴリを制覇
- 間違えた問題は「復習」ボタンで重点学習
- 正答率70%以上を目指す

#### 2. 鳥獣図鑑

```
鳥獣図鑑ボタン → 種類を選択 → 写真と特徴を学ぶ
```

**ポイント**:
- 狩猟鳥獣（〇）と非狩猟鳥獣（×）を確実に見分ける
- 「似ている鳥獣」を比較しながら覚える
- フラッシュカード機能で暗記

#### 3. 模擬試験

```
模擬試験ボタン → 試験開始 → 90分以内に30問解答 → 採点・結果確認
```

**ポイント**:
- 本番と同じ形式・時間制限
- 70%以上（21問正解）で合格判定
- 3回分の模擬試験を全て受ける

#### 4. 実技ガイド

```
実技ガイドボタン → 銃器/わな選択 → 手順を学ぶ
```

**ポイント**:
- 実技試験は「減点方式」
- 安全点検の手順を完璧に覚える
- よくある失敗例を確認

#### 5. 進捗ダッシュボード

```
進捗ボタン → 学習状況・正答率・合格予測を確認
```

**ポイント**:
- カテゴリ別の弱点が一目で分かる
- 合格可能性がリアルタイムで更新される
- 学習時間と解答数を記録

#### 6. 学習ノート

```
学習ノートボタン → メモ作成 → 苦手問題リスト作成
```

**ポイント**:
- 覚えにくい情報をメモ
- 暗記カード機能で復習
- エクスポート機能でバックアップ

## 🎯 よくある質問（FAQ）

### Q1. どのくらい勉強すれば合格できる？

**A**: 目安は**合計20時間以上**の学習です。

- 問題演習: 全115問を3周（10時間）
- 鳥獣図鑑: 全43種を覚える（3時間）
- 実技ガイド: 手順を暗記（2時間）
- 模擬試験: 3回（4.5時間）
- 復習: 間違えた問題の見直し（適宜）

### Q2. 学習の順番は？

**A**: 以下の順序がおすすめです。

```
1週目: 法令 → 猟具 → 基礎知識固め
2週目: 鳥獣識別 → 保護管理 → 実技ガイド
3週目: 模擬試験3回 → 弱点復習 → 本番へ
```

### Q3. スマホだけで学習できる？

**A**: はい、**スマホのみで完結**します。

- PWA対応でアプリのように使える
- オフラインでも全機能が使える
- 通勤・通学中のすきま時間に最適

### Q4. データは消えない？

**A**: ブラウザに保存されますが、以下に注意してください。

- ✅ 同じブラウザなら消えない
- ⚠️ ブラウザのデータ削除で消える
- ⚠️ 別デバイスとは同期しない（現在）

**対策**: 学習ノートのエクスポート機能でバックアップ

### Q5. 模擬試験はいつ受けるべき？

**A**: 以下のタイミングがおすすめです。

| 回数 | タイミング | 目標スコア |
|------|-----------|-----------|
| **1回目** | 2週間学習後 | 60%（18問正解） |
| **2回目** | 3週間学習後 | 70%（21問正解） |
| **3回目** | 試験前日 | 80%（24問正解） |

### Q6. 実技試験が不安…

**A**: 実技ガイドで徹底対策できます。

- 銃器・わなの取り扱い手順（動画想定の詳細解説）
- 安全点検チェックリスト
- よくある失敗例と対策
- 減点ポイントの解説

**ポイント**: 実技は「落ち着いて手順通り」が最重要

## 📞 困ったときは

### トラブルシューティング

| 問題 | 解決方法 |
|------|---------|
| アプリが開かない | ブラウザを最新版に更新 |
| データが消えた | バックアップから復元（学習ノートのインポート機能） |
| 音が出ない | 設定画面で音量・効果音をONに |
| 画面が崩れる | ブラウザのキャッシュをクリア |
