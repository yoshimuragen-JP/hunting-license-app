## 🛠️ トラブルシューティング

### ポートがすでに使われている

**症状**: `Address already in use` エラーが出る

**解決方法**: 別のポートを使う

```bash
python3 -m http.server 3000  # ポート3000で起動
```

### アプリが開かない

**症状**: ブラウザで開いても真っ白

**解決方法**:
1. サーバーが起動しているか確認（ターミナルを見る）
2. URLが正しいか確認（`http://localhost:8000`）
3. ブラウザのコンソールでエラーを確認（F12キー）

### データファイルが読み込めない

**症状**: 「データ読み込みエラー」と表示される

**解決方法**:
1. データファイルが存在するか確認
   ```bash
   ls ~/projects/surprise/2026-02-06/*.json
   ```
2. パスが正しいか確認（appフォルダから相対パス `../` で親ディレクトリ）

## ❓ よくある質問

### このアプリだけで合格できますか？

**回答**: このアプリは学習補助ツールです。以下を併用することをおすすめします：

1. **狩猟読本**（公式テキスト）
2. **予備講習会**（実技対策に効果的）
3. **都道府県の試験案内**（地域特有の規則確認）

### どのくらい勉強すれば合格できますか？

**推奨学習時間**: 合計20-30時間

- 基礎学習: 5-7時間
- 問題演習: 12-18時間（220問以上）
- 実技対策: 3-5時間
- 模擬試験: 2-3時間

ただし、個人差があります。模擬試験で80%以上取れるまで学習を続けることをおすすめします。

### 学習データをバックアップしたい

**回答**: v1.1.0でデータエクスポート/インポート機能を追加しました。

1. ダッシュボードページを開く
2. 「データ管理」セクションへ移動
3. 「エクスポート」ボタンで学習データをダウンロード
4. デバイス変更時は「インポート」でデータ復元

デバイス間でのデータ移行や、定期的なバックアップに便利です。

### オフラインでも使えますか？

**回答**: はい。PWAとしてインストールすれば、オフラインでも使えます。

1回目の起動時にオンラインで開く
→ Service Workerがデータをキャッシュ
→ 2回目以降はオフラインでも動作

### スマホでも使えますか？

**回答**: はい。モバイル最適化されています。

- レスポンシブデザイン
- タッチ操作対応
- PWAとしてホーム画面に追加可能
