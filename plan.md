# 狩猟免許試験（第一種）一発合格学習教材 企画書

## 1. コンセプト

### 学習体験の設計思想

**「ゲーム感覚で楽しく、でも本気で受かる」**

狩猟免許試験は「適性試験」「知識試験」「実技試験」の3つから構成され、特に実技試験は落ちる人の9割を占める最大の難関。しかし従来の学習方法は：
- 分厚い狩猟読本を読むだけ
- 実技の練習ができない
- 引っかけ問題への対策が不十分

この教材は、**完全初心者が独学でも一発合格できる**ことを目指します。

### 核心的な価値提案

1. **実技試験の完全シミュレーション**
   - 3D/アニメーションで猟具の操作を視覚的に学べる
   - 減点ポイントを明示し、「何をしたら減点か」を事前に把握
   - 繰り返し練習できるため、本番で失敗しない

2. **引っかけ問題データベース**
   - 過去問パターンを徹底分析
   - 「よくある間違い」を明示
   - 解説で「なぜ引っかけなのか」を理解

3. **鳥獣判別の特訓**
   - 狩猟読本だけでは不十分な鳥獣識別を強化
   - 写真・イラスト・特徴の3点セットで記憶定着
   - クイズ形式で楽しく反復学習

4. **学習進捗の可視化**
   - RPG風レベルシステム
   - 「弱点分野」を自動検出
   - 達成感を感じながら学習継続

### 差別化ポイント

| 従来の学習方法 | この教材 |
|--------------|---------|
| 狩猟読本を読む（受動的） | インタラクティブなクイズ・シミュレーション（能動的） |
| 実技練習は予備講習会のみ | 何度でも実技をシミュレーション |
| 引っかけ問題に気づかない | 引っかけパターンを明示 |
| 進捗が見えない | レベル・達成度を可視化 |
| 複数の教材が必要 | これ1つで完結 |

## 2. 狩猟免許試験の全体像

### 試験の種類

| 免許の種類 | 使用できる猟具 |
|-----------|--------------|
| **第一種銃猟** | 装薬銃（散弾銃・ライフル銃）、空気銃、わな、網 |
| 第二種銃猟 | 空気銃、わな、網 |
| わな猟 | わな |
| 網猟 | 網 |

**第一種が最も幅広い猟具を使用できる**ため、本教材はこれをターゲットとします。

### 試験科目と合格基準

#### 1. 知識試験（筆記試験）

- **形式**: 三肢択一式（マークシート）
- **問題数**: 30問
- **制限時間**: 90分
- **合格基準**: 70%以上（21問以上正解）

**出題分野の内訳：**

| 分野 | 問題数 | 内容 |
|-----|-------|------|
| **鳥獣保護管理法** | 13問 | 法律の目的、狩猟鳥獣、禁止事項、狩猟期間、狩猟免許制度、登録制度 |
| **猟具の知識** | 6問 | 銃の構造、取り扱い、点検方法、安全管理 |
| **鳥獣の知識** | 9問 | 狩猟鳥獣と非狩猟鳥獣の識別、生態、特徴 |
| **鳥獣保護管理** | 2問 | 保護管理の考え方、狩猟の役割 |

#### 2. 適性試験

| 項目 | 基準（第一種銃猟） |
|-----|------------------|
| **視力** | 両眼で0.7以上、かつ片眼それぞれ0.3以上。片眼が0.3未満または見えない場合、他眼の視野が左右150度以上で視力0.7以上 |
| **聴力** | 10メートルの距離で90デシベルの警音器の音が聞こえること |
| **身体能力** | 銃猟を安全に行うのに支障を及ぼす身体の障害がないこと（補助具使用可） |

#### 3. 実技試験（技能試験）

- **採点方式**: 100点満点から減点方式
- **合格基準**: 70点以上（減点30点以下）
- **重要**: **実技試験で落ちる人が9割**

**試験内容：**

| 項目 | 内容 | 主な減点ポイント |
|-----|------|----------------|
| **鳥獣判別** | 狩猟鳥獣と非狩猟鳥獣を識別（イラスト or 写真） | 誤判別は1つにつき減点大（狩猟読本にない鳥獣が出ることも） |
| **猟銃の点検・分解・結合** | 銃の安全確認、分解・組み立て | 銃口を人に向ける、安全確認を怠る |
| **猟銃の取り扱い動作** | 携行姿勢、休憩時の置き方、受け渡し | 銃口管理の失敗、引き金に指をかける |
| **装填・射撃姿勢** | 弾の装填、射撃姿勢 | 装填手順のミス、不安定な姿勢 |
| **距離の目測** | 距離を目測で判定 | 大幅な誤差 |

### 難易度と対策のポイント

#### 知識試験：中程度
- **合格率**: 予備講習会を受ければ9割以上
- **対策**: 引っかけ問題に注意。問題文をよく読む冷静さが必要
- **学習時間**: 初心者講習会を受けた上で10〜20時間

#### 適性試験：易しい
- **対策**: 視力・聴力に問題がなければほぼ確実に合格
- **注意**: 第一種は視力基準が厳しいため、眼鏡・コンタクトで対応

#### 実技試験：非常に難しい
- **合格率**: 不合格者の9割がここで落ちる
- **対策**:
  - **減点方式のため、いかにミスを減らすかが勝負**
  - 予備講習会での実技練習が必須とされている
  - 本教材では「シミュレーション」で事前に減点ポイントを把握

**最大の山場は「鳥獣判別」と「猟銃の取り扱い」**
- 鳥獣判別：狩猟読本丸暗記では不十分。写真やイラストが多様
- 猟銃取り扱い：銃口管理、安全確認を1回でも忘れると大減点

### 学習フロー（ゼロ→合格まで）

#### Phase 1: 基礎固め（1週間）
1. **狩猟免許とは？** - 試験の全体像を理解
2. **鳥獣保護管理法の基礎** - 法律の目的・狩猟鳥獣・禁止事項
3. **猟具の基礎** - 銃の構造・名称を覚える
4. **鳥獣の基礎** - 狩猟鳥獣と非狩猟鳥獣を知る

#### Phase 2: 知識試験対策（2週間）
1. **過去問演習** - 分野別に過去問を解く
2. **引っかけ問題特訓** - よくある引っかけパターンを学ぶ
3. **弱点克服** - 正答率の低い分野を重点的に学習
4. **総合模擬試験** - 本番形式で力試し

#### Phase 3: 実技試験対策（1週間）
1. **鳥獣判別特訓** - 写真・イラストで繰り返し練習
2. **猟銃取り扱いシミュレーション** - 3D動画で手順を覚える
3. **減点ポイント確認** - 何をしたら減点かを徹底理解
4. **実技模擬試験** - 本番形式でシミュレーション

#### Phase 4: 総仕上げ（試験前日〜当日）
1. **最終確認チェックリスト** - 重要事項の総復習
2. **試験当日の流れ確認** - 持ち物・心構え
3. **よくある失敗集** - 落とし穴を再確認

## 4. コンテンツ詳細

### 知識試験対策

#### A. 鳥獣保護管理法（13問）

**学習内容：**

1. **法律の目的・定義**
   - 鳥獣保護管理法の目的は何か
   - 「鳥獣」「狩猟鳥獣」の定義

2. **狩猟鳥獣**
   - 狩猟できる鳥獣（鳥類28種、獣類20種）
   - 狩猟できない鳥獣

3. **狩猟期間**
   - 一般的な狩猟期間（11月15日〜2月15日）
   - 北海道の特例
   - 都道府県による変更

4. **狩猟禁止・制限**
   - 狩猟禁止場所（鳥獣保護区、公道、住宅密集地など）
   - 狩猟禁止の方法（夜間、薬物使用、動力船使用など）
   - 制限事項

5. **狩猟免許制度**
   - 免許の種類
   - 有効期限（3年）
   - 更新手続き

6. **狩猟者登録制度**
   - 登録の必要性
   - 登録手続き

**引っかけ問題パターン：**

- 「すべて正しい」「すべて誤り」の選択肢に注意
- 数字の微妙な違い（例：狩猟期間の日付）
- 「原則」と「例外」の混同
- 「できる」と「しなければならない」の違い

#### B. 猟具の知識（6問）

**学習内容：**

1. **銃の構造・名称**
   - 散弾銃の各部名称
   - ライフル銃の各部名称
   - 空気銃の各部名称

2. **銃の種類と特徴**
   - 上下二連銃、水平二連銃、自動銃、ボルトアクション
   - それぞれの長所・短所

3. **弾の種類**
   - 散弾（No.の違い）
   - ライフル弾
   - 空気銃弾

4. **点検・整備**
   - 使用前の点検項目
   - 使用後の清掃
   - 保管方法

5. **安全管理**
   - 銃口管理の鉄則
   - 引き金への指のかけ方
   - 装填のタイミング

**実技と連動：**
- ここで学んだ知識が実技試験で必要になる
- 3D動画で視覚的に補強

#### C. 鳥獣の知識（9問）

**学習内容：**

1. **狩猟鳥獣（鳥類28種）**
   - カモ科（マガモ、カルガモ、コガモなど）
   - キジ科（キジ、ヤマドリ）
   - ハト科（キジバト）
   - カラス科（ハシブトガラス、ハシボソガラス）
   - その他

2. **狩猟鳥獣（獣類20種）**
   - イノシシ、ニホンジカ、ツキノワグマなど

3. **非狩猟鳥獣（絶対に間違えてはいけない）**
   - 天然記念物（ニホンカモシカ、ライチョウなど）
   - 絶滅危惧種
   - 外見が似ている種の見分け方

**鳥獣判別の戦略：**

| 学習方法 | 内容 |
|---------|------|
| **写真・イラスト** | 狩猟読本 + 追加の写真データベース |
| **特徴暗記カード** | 色・大きさ・模様・生息地などを箇条書き |
| **類似種比較** | 間違えやすい種を並べて比較 |
| **クイズ形式** | ランダム出題で繰り返し練習 |

**重要：** 狩猟読本のイラストだけでなく、写真や他のイラストも出題される可能性があるため、「特徴で判断する力」を養う。

#### D. 鳥獣保護管理（2問）

**学習内容：**

1. **鳥獣保護管理の考え方**
   - 個体数管理の必要性
   - 生態系のバランス

2. **狩猟の役割**
   - 鳥獣被害の軽減
   - 個体数調整

### 適性試験対策

**視力・聴力・身体能力の基準を明示**
- 視力は両眼0.7以上、片眼0.3以上
- 聴力は10m先の90dBの音が聞こえること
- 身体能力は補助具使用可

**対策：**
- 視力が不安な場合は眼鏡・コンタクトで対応
- 聴力検査は病院で事前確認可能
- 身体障害がある場合は補助具の使用を事前相談

---

### 実技試験対策

#### 1. 鳥獣判別シミュレーション

**機能：**
- **写真モード**: リアルな写真で判別練習
- **イラストモード**: 狩猟読本風イラストで練習
- **特徴暗記カード**: フラッシュカード形式で特徴を暗記
- **類似種比較**: 間違えやすい種を並べて表示

**減点ルール：**
- 狩猟鳥獣を「狩猟不可」と判定 → 大減点
- 非狩猟鳥獣を「狩猟可」と判定 → **さらに大減点（最悪一発アウト）**

**出題傾向：**
- 狩猟読本に載っていない写真・イラストが出ることがある
- 「特徴」で判断する力が必要

#### 2. 猟銃取り扱いシミュレーション

**A. 点検・分解・結合**

- **3D動画**で手順を視覚化
- 各ステップの注意点を表示
- 「減点ポイント」を明示

**主な減点ポイント：**
- 銃口を人に向ける（大減点）
- 安全確認を怠る（大減点）
- 分解・結合の手順ミス
- 銃の取り扱いが雑

**B. 携行・休憩時・受け渡し**

- **アニメーション**で正しい動作を学習
- 間違った動作も表示し、「これはNG」と明示

**主な減点ポイント：**
- 銃口管理の失敗（大減点）
- 引き金に指をかける（大減点）
- 受け渡し時の不注意

**C. 装填・射撃姿勢**

- **アニメーション**で正しい姿勢を学習
- 「この姿勢だと不安定」というNGパターンも表示

**主な減点ポイント：**
- 装填手順のミス
- 不安定な射撃姿勢
- 安全確認を怠る

**D. 距離の目測**

- **トレーニングモード**で距離感を養う
- 正解との誤差を表示
- 繰り返し練習で精度向上

#### 3. 実技試験の心構え

**絶対に守るべきルール：**
1. **銃口は常に安全な方向へ**
2. **引き金に指をかけない（装填・射撃時以外）**
3. **安全確認は毎回必ず行う**
4. **慌てない・焦らない**
5. **丁寧な動作を心がける**

**減点方式の特性：**
- 100点から減点されていく
- 30点以上減点で即不合格
- **1つの大きなミスで不合格もあり得る**

**対策：**
- 「やってはいけないこと」を徹底的に覚える
- 正しい手順を体に染み込ませる
- 本番は緊張するため、何度も練習して自動化

### 過去問・模擬試験

#### 知識試験 模擬試験

- **本番形式**: 30問・90分
- **自動採点**: 即座に結果表示
- **詳細な解説**: 各問題の解説 + 引っかけポイント
- **分野別正答率**: 弱点を可視化
- **複数回実施**: 複数パターンの模擬試験

#### 実技試験 模擬試験

- **鳥獣判別**: ランダムに10〜15種出題
- **猟銃取り扱い**: シミュレーション形式
- **減点計算**: 自分の減点を確認

## 7. リスクと対策

### リスク1: 鳥獣画像の著作権

**問題**: 鳥獣の写真・イラストを大量に必要とするが、著作権が心配

**対策**:
- **フリー素材サイト**を活用（Unsplash, Pixabay, Pexels）
- **政府・自治体の公開資料**を活用（環境省のWebサイトなど）
- **CC0ライセンスの画像**を優先的に使用
- どうしても見つからない場合は、簡易的なイラストを自作

### リスク2: 実技試験のシミュレーション精度

**問題**: 実技試験の採点基準が公開されていないため、減点の正確性が不明

**対策**:
- **予備講習会の情報**を参考にする
- **合格者の体験談**を収集
- **「一般的な安全ルール」を基準**にする（銃口管理、安全確認など）
- 「これは推定です」と明記し、予備講習会の受講も推奨

### リスク3: 3D/アニメーションの開発コスト

**問題**: Three.jsでの3D表示やアニメーション作成は技術的に難易度が高い

**対策**:
- **Phase 1では簡易版**（静止画 + テキスト説明）で実装
- **Phase 3で段階的にアニメーション追加**
- 3Dモデルは既存のフリー素材を活用
- 最悪の場合、動画埋め込みで代用

### リスク4: コンテンツの正確性

**問題**: 法令や試験内容が変更される可能性

**対策**:
- **環境省・都道府県の公式情報**を参照
- **最新の狩猟読本**を確認
- 「2026年版」と明記し、定期的な更新を推奨
- ユーザーからのフィードバックを受け付ける仕組み

### リスク5: 学習教材としての効果

**問題**: 本当にこれで合格できるか？

**対策**:
- **「予備講習会の受講は推奨」と明記**
- この教材はあくまで「補助教材」として位置づける
- 過去問を多数収録し、試験傾向をカバー
- テストユーザーで実際に受験してもらい、フィードバック収集
