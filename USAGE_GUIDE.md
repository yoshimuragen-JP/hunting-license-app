## 1. アプリの起動方法

### 📱 方法1: クイックスタート（推奨）

**一番速い起動方法**

```bash
cd ~/projects/surprise/2026-02-06
./start.sh
```

ブラウザで `http://localhost:8000` を開く → 完了！

### 🖥️ 方法2: 手動起動

```bash
cd ~/projects/surprise/2026-02-06/app
python3 -m http.server 8000
```

### 📲 方法3: PWAとしてインストール

**スマホにアプリとしてインストールする方法**

#### iPhone
1. Safariでアプリを開く
2. 画面下の共有ボタン（四角に↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ
5. ホーム画面にアイコンが追加される

#### Android
1. Chromeでアプリを開く
2. メニュー（︙）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ

## 2. 基本学習の進め方

### ステップ1: 学習ガイドを読む

まず「学習ガイド」ページで全体像を把握します。

1. トップページの「学習ガイド」をクリック
2. 3週間の学習プランを確認
3. 自分のペースに合わせて調整

### ステップ2: カテゴリを選ぶ

5つのカテゴリから学習したいものを選びます：

| カテゴリ | 内容 | 難易度 |
|---------|------|--------|
| **法令** | 鳥獣保護管理法、狩猟法 | ⭐⭐ |
| **猟具** | 銃器、わな、網の知識 | ⭐⭐⭐ |
| **鳥獣識別** | 狩猟鳥獣と非狩猟鳥獣の見分け方 | ⭐⭐⭐⭐ |
| **保護管理** | 鳥獣保護区、狩猟期間 | ⭐⭐ |
| **実技** | 銃器の取り扱い、安全管理 | ⭐⭐⭐ |

### ステップ3: 自分のペースで学ぶ

- **1日30分〜1時間**: 無理なく続けられる
- **週3回以上**: 記憶の定着に効果的
- **毎日同じ時間**: 習慣化しやすい

## 3. 問題演習の使い方

### 基本的な流れ

1. カテゴリまたは「ランダム」を選択
2. 問題数を選ぶ（5問/10問/20問）
3. 問題を解く
4. 解説を読む
5. 次の問題へ

### 効果的な学習法

#### ✅ 正解した問題も解説を読む
「なぜ正解か」を理解することで、記憶が定着します。

#### ✅ 間違えた問題はメモする
「学習ノート」機能で、間違えた理由をメモしましょう。

#### ✅ 復習機能を活用する
間違えた問題は「復習リスト」に自動で追加されます。

## 4. 模擬試験の受験方法

### 受験前の準備

- **時間**: 90分確保する
- **環境**: 静かな場所で集中できる環境
- **準備**: 筆記用具、メモ用紙を用意

### 受験の流れ

1. トップページの「模擬試験」をクリック
2. 試験番号（1〜3回）を選択
3. 「試験開始」をクリック
4. 30問を90分以内に解答
5. 「採点」をクリック
6. 結果と解説を確認

### 結果の見方

| 正答率 | 評価 |
|--------|------|
| 90%以上 | 優秀 - 合格圏内 |
| 80-89% | 良好 - もう少し |
| 70-79% | 合格ライン - 弱点を克服 |
| 70%未満 | 要復習 - 基礎から |

## 5. 鳥獣図鑑の活用法

### 基本的な使い方

1. 「鳥獣図鑑」ページを開く
2. 狩猟鳥獣/非狩猟鳥獣のタブを切り替え
3. 各種の詳細情報を確認

### 効果的な覚え方

#### 📸 フラッシュカード機能
1. 「フラッシュカード」ボタンをクリック
2. 画像を見て名前を思い出す
3. クリックして答えを確認
4. 「覚えた」「まだ」で分類

#### 🎯 5秒判別トレーニング
1. ランダムに表示される鳥獣を5秒以内に判別
2. 狩猟鳥獣か非狩猟鳥獣かを選択
3. 即座にフィードバック

## 8. 進捗管理の見方

### ダッシュボード

1. トップページの「ダッシュボード」をクリック
2. 以下の情報を確認：
   - 総合正答率
   - カテゴリ別正答率
   - 学習時間
   - 解答数
   - 合格可能性

### グラフの見方

- **円グラフ**: カテゴリ別の解答数
- **棒グラフ**: カテゴリ別の正答率
- **折れ線グラフ**: 時系列での正答率推移

## 10. よくあるトラブル

### Q: データが読み込めない

**A:** 以下を確認してください：

1. サーバーが起動しているか
2. URLが正しいか（http://localhost:8000）
3. データファイルが存在するか
   ```bash
   ls ~/projects/surprise/2026-02-06/*.json
   ```

### Q: 学習データが消えた

**A:** ブラウザのデータを削除すると、学習データも消えます。

**対策**: 定期的にデータをエクスポート（今後実装予定）

### Q: オフラインで使えない

**A:** 初回アクセス時はインターネット接続が必要です。

1. オンラインで一度アクセス
2. Service Workerがインストールされる
3. 2回目以降はオフラインでも動作

### Q: スマホで動作が遅い

**A:** 以下を試してください：

1. ブラウザのキャッシュをクリア
2. アプリを再起動
3. PWAとしてインストール
