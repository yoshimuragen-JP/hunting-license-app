## 📊 問題データの追加

新しい問題を追加する場合は、以下のフォーマットに従ってください：

```json
{
  "id": "Q066",
  "category": "law",
  "difficulty": "medium",
  "question": "問題文をここに記載します。",
  "options": [
    "選択肢1",
    "選択肢2",
    "選択肢3"
  ],
  "correctAnswer": 0,
  "explanation": "正解の理由を詳しく説明します。",
  "source": "狩猟読本 p.42",
  "tips": "覚え方のコツがあれば記載"
}
```

### 必須項目
- **id**: 一意のID（Q001〜）
- **category**: `law`, `equipment`, `animals`, `management`, `practical`
- **difficulty**: `easy`, `medium`, `hard`
- **question**: 問題文
- **options**: 3つの選択肢
- **correctAnswer**: 正解のインデックス（0-2）
- **explanation**: 解説
