## 問題の構造

各問題は以下の情報を含みます：

```json
{
  "id": "law_001",
  "category": "法令",
  "difficulty": "易",
  "question": "狩猟免許の有効期間は何年ですか？",
  "choices": ["1年", "3年", "5年"],
  "answer": 1,
  "explanation": "狩猟免許の有効期間は3年です。更新手続きが必要です。"
}
```

- **id**: 問題の一意識別子
- **category**: カテゴリ（法令、猟具、鳥獣、鳥獣保護管理、実技）
- **difficulty**: 難易度（易、中、難）
- **question**: 問題文
- **choices**: 選択肢（3つ）
- **answer**: 正解のインデックス（0-2）
- **explanation**: 詳細な解説（100文字以上）

## カテゴリの説明

| カテゴリ | 内容 |
|---------|------|
| **法令** | 鳥獣保護管理法、狩猟者登録、禁止事項、罰則など |
| **猟具** | 銃器、わな、網などの種類、取り扱い、保管方法 |
| **鳥獣** | 狩猟鳥獣の識別、生態、生息環境 |
| **鳥獣保護管理** | 個体数管理、外来種対策、ジビエ利用など |
| **実技** | 銃器の点検、わなの設置、安全管理、鳥獣判別 |

## 難易度の基準

| 難易度 | 説明 |
|--------|------|
| **易** | 基本的な知識、頻出問題 |
| **中** | 実践的な知識、応用問題 |
| **難** | 詳細な知識、複雑な判断が必要な問題 |

## 問題の品質保証

全ての問題は以下の基準を満たしています：

1. **正確性**: research.mdとSOURCES.mdの情報源に基づく
2. **難易度バランス**: 易（20-30%）、中（50%）、難（20-30%）
3. **三肢択一形式**: 全ての問題は選択肢3つ
4. **詳細な解説**: 各問題に100文字以上の解説
5. **実践的**: 実際の試験に出題されそうな内容

## 使用方法

アプリは自動的にこれらのファイルを読み込み、ユーザーの学習レベルに応じて適切な問題を出題します。
